# 财富金额统计与增长应用项目宪法

## 项目概述
本项目旨在开发一个个人财富金额统计与增长分析应用，帮助用户在2026年至2030年五年期限内跟踪、分析和管理个人财务状况。



\## 1. 核心原则

- **安全性第一**：严禁执行任何破坏性 shell 命令（如 rm, drop table）。 - 
- **类型安全**：所有代码必须严格类型化，禁止使用 `any`。

 ## 2. 代码规范

 ##  - 语言：TypeScript (ES2022+)

 ##  - 风格：遵循 Prettier 默认配置，缩进为 2 空格。

 ##  - 命名：变量使用 camelCase，组件使用 PascalCase。 

## 3. 架构约束 

- 禁止直接导入 `@/utils` 以外的内部模块到 `components` 目录。
- 所有 API 请求必须通过 `api/client.ts` 发起，禁止直接使用 fetch。



## 4. 测试要求 

- 任何新功能的代码提交必须包含对应的单元测试。
-  - 测试覆盖率不得低于 80%。







## 核心功能

### 1. 财富记录功能
- 支持记录收入金额及来源
- 支持记录支出金额及用途
- 记录时间戳和备注信息
- 支持多种货币类型（默认人民币）

### 2. 数据可视化功能
- 折线图展示：显示财富金额随时间的变化趋势
- 自定义时间范围：支持每日、每周、每月、每季度、每年的数据展示
- 增长趋势分析：自动计算并显示财富增长率
- 收支对比图表：可视化收入与支出的对比分析

### 3. 用户界面
- 简洁直观的前端页面
- 响应式设计，支持移动端访问
- 数据录入表单
- 图表展示区域
- 数据统计概览面板

### 4. 后端功能
- 使用Python编写
- 数据存储：本地数据库（SQLite或其他轻量级数据库）
- API接口：提供数据的CRUD操作
- 数据处理：计算统计指标和增长率
- 安全性：用户数据本地存储，保障隐私

## 技术架构

### 前端技术栈
- HTML5/CSS3
- JavaScript (ES6+)
- 图表库：Chart.js或ECharts
- 样式框架：Bootstrap或Tailwind CSS

### 后端技术栈
- Python 3.x
- 框架：Flask或FastAPI
- 数据库：SQLite（默认）或PostgreSQL
- ORM：SQLAlchemy

### 部署方式
- 本地运行（Windows/Linux/macOS）
- 可选：Docker容器化部署
- 可选：云端部署（如Vercel、Heroku等）
